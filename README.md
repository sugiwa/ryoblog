# Nextjs + microCMSで作成したブログサイト

GatsbyjsとContentfulで作成したブログをNextjsとmicroCMSで作り直した。

## 仕様技術
- Nextjs
- TypeScript
- Tailwind CSS
- microCMS

## 追加したいこと
- ページネーション
- パンくずリスト
- サイトマップ
